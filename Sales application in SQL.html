Sales Data Analysis using SQL Oracle

In this project, I created a few tables using SQL developer and linked them with foreign keys to create a join between them. In this project, I learned how to create primary keys, how to make foreign keys, how to link two tables using foreign keys, how to alter tables and many more queries DDL, and DML queries.

These are the few questions which i solved using SQL developer,

  • Q. 1 Top 3 Selling Cities.
  • Q. 2 Top 3 Selling Products.
  • Q. 3 Product wise Total Sales.
  • Q. 4 Customers purchase details.
  • Q. 5 Top 3 Categories.
  • Q. 6 Monthly, Daily, and Weekly Sales.
  • Q. 7 State wise Product wise Total Sales.
